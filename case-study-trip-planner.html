Case Study

Trip Planner App

A travel planning interface designed to help users organize activities, track plans, and simplify itinerary management.

Role

Project Manager, UX/UI Designer

Tools

Figma, Javascript, HTML, CSS

Focus

Organization, usability, task flow

  • Overview
  • Problem
  • Users
  • Goals
  • Final UI
  • Outcome
  • Reflection

Overview

The Trip Planner app helps users organize travel activities, manage budgets and schedules, and keep track of important details in one place.

Problem

Planning trips often involves multiple apps, notes, and scattered information. This leads to confusion and difficulty keeping track of plans.

Users

  • Students and young travelers
  • People planning group trips
  • Users who want simple itinerary management

Design Goals

  • Simplify adding and organizing activities
  • Create a clear visual timeline
  • Reduce clutter and confusion
  • Make task management intuitive

Final UI

The final interface emphasizes clarity through spacing, hierarchy, and consistent layout patterns. Key actions are easy to find and use.

Outcome

The design improves organization and reduces cognitive load by centralizing all trip information into a single, easy-to-use interface.

Reflection

Future improvements could include user testing, adding collaboration features, and refining the visual timeline experience.
